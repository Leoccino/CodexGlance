Default view

CodexGlance opens with the 5-hour window only. Weekly usage can be added from the menu when it matters.

Install

For non-technical users.

No terminal required. Download the app, unzip it, move it to Applications, then right-click Open the first time because the app is currently unsigned.

  1. Download Get CodexGlance.app.zip from GitHub Releases.
  2. Move Unzip it and drag CodexGlance.app into Applications.
  3. Open once Right-click CodexGlance.app, choose Open, then confirm Open.
  4. Glance Codex usage appears in the macOS menu bar.
